Pip

Para esto iremos a file en visualCode

y cerramos el folder
Ingresamos a la parte de la terminal y escribimos lo siguente:
herramienta que nosotros vayamos a utilizar, despues indicamos la accion que  este realice, indicarle contra que queremos que realice la accion: Ejemplo
pip install requests

Si en este caso nos da un error como por ejemplo el siguente, debemos de leerlo.
Ejemplo
[notice] A new release of pip available: 22.3.1 -> 23.1.2
[notice] To update, run: python.exe -m pip install --upgrade pip
En este caso nos esta diciendo que debemos de actualizar el pip a la version mas actual
para ver si pip se actualizo correctamente debemos de colocar el siguente comando 
pip --version
en pip list nos muestra todos los paquetes 

para instalar una version anterior debemos de ingresar los siguiente

Herramienta que vayamos a utilizar uninstall y el nombre del paquete
Ejemplo 
pip uninstall requests y presionamos enter y le daremos que si 

Para instalar una version que querramos 

herramienta que nosotros vayamos a utilizar, despues indicamos la accion que  este realice, indicarle contra que queremos que realice la accion==Version: Ejemplo 

pip install requests==2.18.1

Para instalar una version compatible
el * indica una version 
pip install requests==2.18.*

Con esto le decimos que el escoja lo que quiera

pip install requests~=2.18.1
